Linear Algebra Notes

The Subspace Theorem

Theorem: A non-empty subset W of a vector space V is a subspace of V if and only if for all vectors u, v ∈ W and all scalars α, β ∈ F (the field), the linear combination αu + βv is also in W.

Proof

This proof has two parts:

Part 1: The "if" direction (⇒)

Assume W is a subspace of V. We need to show that αu + βv ∈ W.

  • 1. Let u, v ∈ W and scalars α, β ∈ F.
  • 2. Since W is a subspace, it is closed under scalar multiplication. Therefore, αu ∈ W and βv ∈ W.
  • 3. Since W is a subspace, it is also closed under vector addition. Therefore, the sum of αu and βv must be in W.
  • 4. Thus, αu + βv ∈ W.

Part 2: The "only if" direction (⇐)

Assume αu + βv ∈ W for all u, v ∈ W and α, β ∈ F. We must show that W is a subspace by verifying it satisfies the necessary properties.

  • A1: Closure under Addition: Let u, v ∈ W. Choose α=1 and β=1. By our assumption, (1)u + (1)v = u + v ∈ W. The set is closed under addition.
  • A3: Existence of Zero Vector: Since W is non-empty, let u be some vector in W. Choose α=0 and β=0. Then (0)u + (0)u = 0 + 0 = 0 ∈ W. The zero vector is in W.
  • A4: Existence of Additive Inverse: Let u ∈ W. Choose α=-1 and β=0. Then (-1)u + (0)u = -u ∈ W. For every vector in W, its additive inverse is also in W.
  • M1: Closure under Scalar Multiplication: Let u ∈ W and α be any scalar. Choose β=0. Then αu + (0)u = αu ∈ W. The set is closed under scalar multiplication.

Since W is a subset of V, other properties like associativity and distributivity are inherited from V. With closure under addition and scalar multiplication, and the existence of the zero vector, W is a subspace of V.

Sum of Subspaces

Definition: Let W₁ and W₂ be subspaces of a vector space V. The sum of these subspaces, denoted W₁ + W₂, is the set of all possible sums of their elements:

W₁ + W₂ = {w₁ + w₂ | w₁ ∈ W₁, w₂ ∈ W₂}

Theorem: The sum W₁ + W₂ is also a subspace of V.

Proof

  • 1. Non-empty: Since W₁ and W₂ are subspaces, they both contain the zero vector 0. Thus, 0 = 0 + 0 ∈ W₁ + W₂, so the set is not empty.
  • 2. Closure under Addition: Let u = u₁ + u₂ and v = v₁ + v₂ be two vectors in W₁ + W₂, where u₁, v₁ ∈ W₁ and u₂, v₂ ∈ W₂.
    u + v = (u₁ + u₂) + (v₁ + v₂) = (u₁ + v₁) + (u₂ + v₂).
    Since W₁ and W₂ are subspaces, (u₁ + v₁) ∈ W₁ and (u₂ + v₂) ∈ W₂. Therefore, u + v ∈ W₁ + W₂.
  • 3. Closure under Scalar Multiplication: Let u = u₁ + u₂ ∈ W₁ + W₂ and α be a scalar.
    αu = α(u₁ + u₂) = αu₁ + αu₂.
    Since W₁ and W₂ are subspaces, αu₁ ∈ W₁ and αu₂ ∈ W₂. Therefore, αu ∈ W₁ + W₂.

Since W₁ + W₂ is non-empty and closed under both addition and scalar multiplication, it is a subspace of V.

Examples: Identifying Subspaces

Example 1: Is W = {(a,b,c) | a ≤ b ≤ c} a subspace of R³?

1. Zero Vector: The vector (0,0,0) is in W because 0 ≤ 0 ≤ 0. So W is non-empty.

2. Closure under Addition: Let u=(a,b,c) and v=(d,e,f) be in W. Then a ≤ b ≤ c and d ≤ e ≤ f.
u+v = (a+d, b+e, c+f).
Since a ≤ b and d ≤ e, then a+d ≤ b+e.
Since b ≤ c and e ≤ f, then b+e ≤ c+f.
So, a+d ≤ b+e ≤ c+f, and W is closed under addition.

3. Closure under Scalar Multiplication: Let u=(1,2,3) ∈ W. Let the scalar α = -1.
αu = -1 * (1,2,3) = (-1,-2,-3).
To be in W, we must have -1 ≤ -2 ≤ -3. This is false.

Conclusion: W is not closed under scalar multiplication, so it is not a subspace of R³.

Example 2: Is W = {(a,b,c) | b + 4c = 0} a subspace of R³?

The condition b + 4c = 0 means b = -4c. So any vector in W has the form (a, -4c, c) for any a, c ∈ R.

1. Zero Vector: If a=0 and c=0, we get (0,0,0). So W is non-empty.

2. Closure under Addition: Let u=(a₁, -4c₁, c₁) and v=(a₂, -4c₂, c₂) be in W.
u+v = (a₁+a₂, -4c₁-4c₂, c₁+c₂) = (a₁+a₂, -4(c₁+c₂), c₁+c₂).
This vector is of the form (a', -4c', c') where a'=a₁+a₂ and c'=c₁+c₂. It satisfies the condition. W is closed under addition.

3. Closure under Scalar Multiplication: Let u=(a, -4c, c) ∈ W and α be a scalar.
αu = α(a, -4c, c) = (αa, -4(αc), αc).
This vector is of the form (a', -4c', c') where a'=αa and c'=αc. It satisfies the condition. W is closed under scalar multiplication.

Conclusion: W is closed under both operations, so it is a subspace of R³.

Linear Combinations and Span

Linear Combination: A vector v is a linear combination of vectors {v₁, v₂, ..., vₙ} if it can be written as v = α₁v₁ + α₂v₂ + ... + αₙvₙ for some scalars {α₁, α₂, ..., αₙ}.

Example: Express v = (1, 7, -4) as a linear combination of v₁ = (1, -3, 2) and v₂ = (2, -1, 1).

We want to find α₁ and α₂ such that v = α₁v₁ + α₂v₂.

(1, 7, -4) = α₁(1, -3, 2) + α₂(2, -1, 1)

This gives a system of linear equations:

  • (1) α₁ + 2α₂ = 1
  • (2) -3α₁ - α₂ = 7
  • (3) 2α₁ + α₂ = -4

Solving this system gives α₁ = -3 and α₂ = 2.

Result: v = -3v₁ + 2v₂

Linear Span: The linear span of a set of vectors S, denoted L(S) or Span(S), is the set of all possible linear combinations of the vectors in S. This set forms a vector subspace.

Linear Independence and Basis

Linearly Independent: A set of vectors {v₁, ..., vₙ} is linearly independent if the only solution to the equation α₁v₁ + ... + αₙvₙ = 0 is the trivial solution, where all scalars αᵢ = 0.
Linearly Dependent: If there is at least one non-zero scalar in the solution, the set is linearly dependent.
Basis: A basis for a vector space V is a set of vectors that is both:
  1. Linearly independent.
  2. Spans the entire vector space V.

Example: Is S = {(1,1,2), (1,2,5), (5,3,4)} a basis for R³?

We check for linear independence first. Set a linear combination to zero:

α₁(1,1,2) + α₂(1,2,5) + α₃(5,3,4) = (0,0,0)

This gives the system of equations:

  • (1) α₁ + α₂ + 5α₃ = 0
  • (2) α₁ + 2α₂ + 3α₃ = 0
  • (3) 2α₁ + 5α₂ + 4α₃ = 0

Subtracting (1) from (2) gives: α₂ - 2α₃ = 0, so α₂ = 2α₃.

Substitute this into (1): α₁ + (2α₃) + 5α₃ = 0, so α₁ = -7α₃.

If we choose α₃ = 1, then we get a non-trivial solution α₁ = -7, α₂ = 2, α₃ = 1.

Conclusion: Since a non-trivial solution exists, the set is linearly dependent and therefore cannot be a basis for R³.

Verification of Vector Space Axioms

Example: Show that V = {(x₁, 0, x₂) | x₁, x₂ ∈ R} is a vector space.

We must verify all the vector space axioms for addition and scalar multiplication.

Additive Axioms

  • A1 (Closure): (x₁, 0, x₂) + (y₁, 0, y₂) = (x₁ + y₁, 0, x₂ + y₂). The result is in V. ✓
  • A2 (Associativity): [(x₁,0,x₂) + (y₁,0,y₂)] + (z₁,0,z₂) = (x₁+y₁+z₁, 0, x₂+y₂+z₂).
    (x₁,0,x₂) + [(y₁,0,y₂) + (z₁,0,z₂)] = (x₁+y₁+z₁, 0, x₂+y₂+z₂). Holds. ✓
  • A3 (Additive Identity): The zero vector is e = (0,0,0), which is in V, since (x₁, 0, x₂) + (0,0,0) = (x₁, 0, x₂). ✓
  • A4 (Additive Inverse): For u = (x₁, 0, x₂), the inverse is -u = (-x₁, 0, -x₂), which is in V. ✓

Multiplicative Axioms

  • M1 (Closure): α(x₁, 0, x₂) = (αx₁, 0, αx₂). The result is in V. ✓
  • M2 (Associativity): α(β(x₁,0,x₂)) = α(βx₁, 0, βx₂) = ((αβ)x₁, 0, (αβ)x₂).
    (αβ)(x₁,0,x₂) = ((αβ)x₁, 0, (αβ)x₂). Holds. ✓
  • M3 (Distributivity): α((x₁,0,x₂)+(y₁,0,y₂)) = α(x₁+y₁, 0, x₂+y₂) = (αx₁+αy₁, 0, αx₂+αy₂).
    α(x₁,0,x₂) + α(y₁,0,y₂) = (αx₁,0,αx₂) + (αy₁,0,αy₂) = (αx₁+αy₁, 0, αx₂+αy₂). Holds. ✓
  • M4 (Distributivity): (α+β)(x₁,0,x₂) = ((α+β)x₁, 0, (α+β)x₂) = (αx₁+βx₁, 0, αx₂+βx₂).
    α(x₁,0,x₂) + β(x₁,0,x₂) = (αx₁,0,αx₂) + (βx₁,0,βx₂) = (αx₁+βx₁, 0, αx₂+βx₂). Holds. ✓
  • M5 (Multiplicative Identity): 1 * (x₁, 0, x₂) = (1*x₁, 0, 1*x₂) = (x₁, 0, x₂). Holds. ✓

Conclusion: All axioms hold, so V is a vector space.
